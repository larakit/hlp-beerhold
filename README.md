#Helper beerhold.it

В twig-шаблонах станут доступными следующие функции

### Цветная фотография (ширина, высота)

~~~
{{ beerhold(600,400) }}
~~~

### Черно-белая фотография (ширина, высота)

~~~
{{ beerhold_g(600,400) }}
~~~

### Sepia (ширина, высота)

~~~
{{ beerhold_s(600,400) }}
~~~

### Негатив (ширина, высота)

~~~
{{ beerhold_n(600,400) }}
~~~
